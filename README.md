## Supported things
- Loading and Saving dialogues
- 4 types of Nodes
    - Single Dialogue Node
    - Options Dialogue Node
    - Start/End Node
    - Grouping Block (unfinished)
- Character creation via ExposedProperties
- Runtime Support
    - Easy to use Manager
    - Demo Scene
    - Typewriter Effect
